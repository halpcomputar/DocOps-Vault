# Internal Operations

Cross-client knowledge lives here.

- `people/` - durable context about coworkers, collaborators, and internal stakeholders
- `vendors/` - reusable vendor context and escalation paths
- `orgs/` - external organizations that are not client-specific vendors
- `projects/` - internal initiatives
- `operations/` - cross-client operational notes
- `security/` - internal security standards and reviews
- `sops/` - procedures meant to be reused
