# Quicknotes

Use this folder for mobile, tablet, or on-the-go text capture that is not ready to be filed.

This is source material, not durable note storage. Quicknotes should eventually be routed into the structured vault or ignored intentionally.

Recommended filename:

`YYYY-MM-DD-<slug>.md`

Suggested quick format:

```markdown
---
created: YYYY-MM-DD
source_type: quicknote
status: inbox
---

# Quicknote

Client:
Location:
Context:

Notes:

Follow-up:
```
