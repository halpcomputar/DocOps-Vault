# Daily Process

Use this prompt to extract durable notes from a daily note without destroying the daily record.

## Prompt

Read `daily/YYYY-MM-DD.md` and propose structured extractions.

Classify each item as:

- client project
- client issue
- incident
- internal operation
- personal project
- personal event
- standing reference update
- leave in daily

Do not move or delete daily note content. Add links or create structured notes only after the vault owner approves the proposed routing.
