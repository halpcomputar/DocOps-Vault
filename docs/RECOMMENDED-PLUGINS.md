# Recommended Obsidian Plugins

DocOps Vault does not require plugins to work, but these are useful.

## Core Obsidian

- Daily notes
- Templates or Templater
- Backlinks
- Search

## Community Plugins

- Tasks - useful for due dates and action tracking
- Dataview or Bases - useful for querying notes by metadata
- Calendar - useful with daily notes
- Style Settings - useful if your theme exposes settings
- Iconic or similar - useful for folder icons and colors
- Local REST API - optional, useful for automation

## Visual Setup

Themes and visual plugins should remain optional. The vault model depends on folders, filenames, links, frontmatter, and ingest prompts, not on a specific theme.

## Sync

Use whichever sync tool you trust:

- Obsidian Sync
- LiveSync
- Git for a private personal vault
- cloud storage, if you understand the conflict risks

Do not sync `.git/` through Obsidian-oriented sync tools.
